# Primary Sources

This directory contains annotated primary source documents for the course "Borderlands in Transition: The U.S.-Mexican War and Native Sovereignty."

Each source is presented as a Markdown file with:

1. **Introduction**: Context about the document's origin, author, and historical moment.
2. **Transcribed Text**: The primary source text (or excerpt) with clear formatting.
3. **Instructor's Annotation**: Analysis linking the source to course themes, Brian DeLay's arguments, and Ford Madox Ford's narrative techniques.
4. **Discussion Questions**: Prompts for class discussion or reflection.

Sources are organized thematically and correspond to weekly modules in the syllabus.
